What is Bootstrap?

Bootstrap is a popular front-end CSS framework used for designing responsive and mobile-first websites. It provides ready-made components such as buttons, forms, cards, navigation bars, and grid layouts. Instead of writing styles from scratch, you can use Bootstrap classes to quickly build professional UI designs.

It comes with a powerful grid system that helps you align, structure, and control your website layout across different screen sizes. Bootstrap is widely used because it is easy to learn, customizable, and works well for both beginners and experienced developers.

Example Usage:

<button class="btn btn-primary">
  Click Me
</button>

Bootstrap helps you build responsive and modern websites quickly and efficiently.
